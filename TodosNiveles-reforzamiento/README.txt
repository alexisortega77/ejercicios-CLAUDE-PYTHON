## Estructura de archivos

    nivel_1_ejercicios.py  → Fundamentos (variables, tipos, operaciones)
    nivel_2_ejercicios.py  → Control de flujo (if, for, while)
    nivel_3_ejercicios.py  → Funciones (def, return, lambda)
    nivel_4_ejercicios.py  → Estructuras de datos (listas, dicts, sets)
    nivel_5_ejercicios_preview.py → POO — resolver después del Nivel 5


## Niveles de dificultad por ejercicio

    Ejercicios 1-3 → Fácil
    Ejercicios 4-6 → Medio
    Ejercicios 7-8 → Desafío


## Cómo trabajar estos ejercicios

    1. Crea un archivo separado por ejercicio
       Ejemplo: nivel_1_ejercicio_1.py

    2. Lee el output esperado antes de escribir código

    3. Sigue el proceso:
       → Que funcione primero (versión larga)
       → Que sea clara (buenos nombres de variables)
       → Que sea óptima (comprehensions, funciones built-in)

    4. Verifica que tu output coincida exactamente con el esperado

    5. Si te bloqueas en el paso 1, consulta la guía de referencia


## Reglas importantes

    ✅ Usa funciones en niveles 3 y 4
    ✅ Las funciones de cálculo siempre usan return
    ✅ El print va fuera de las funciones
    ✅ Nombres de variables en snake_case
    ✅ Constantes en MAYUSCULAS
    ❌ No uses estructuras de niveles superiores
       (no uses listas en nivel 1, no uses clases en nivel 4)


## Consejo

    No intentes hacer la versión óptima desde el inicio.
    Primero que funcione, luego que sea bonita.
    Un código feo que funciona vale más que uno bonito que no terminas.
